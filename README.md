# 🚀 Online Compiler Web App (Full Stack)

🔗 **Live Demo:** https://online-compiler-zeta-eight.vercel.app/editor  

---

## ✨ Features

- 💻 Multi-language Code Execution  
- 🔐 JWT Authentication & Authorization  
- 🔑 Google Authentication Support  
- 📂 Save & Manage Code Snippets  
- ⚡ Real-time Code Execution  
- ☁️ Cloud Deployment  
- 📱 Fully Responsive UI  

---

## 🏗️ Tech Stack

### 🔹 Frontend
- React.js (Vite)
- Tailwind CSS
- JavaScript (ES6+)

### 🔹 Backend
- Spring Boot (Java)
- Spring Security + JWT
- MongoDB
- REST APIs

### 🔹 Deployment
- Vercel (Frontend)
- Spring Boot Server (Backend)

---

## 🔐 Authentication Flow

1. User registers / logs in  
2. JWT token is generated  
3. Token is used for secured API requests  
4. Google OAuth supported  

---

## ⚡ API Endpoints

### 🔑 Auth APIs
- `/api/auth/login`
- `/api/auth/register`
- `/api/auth/google`

### ▶️ Code Execution
- `/api/execute`

### 📂 Snippets
- `/api/snippets`
- `/api/snippets/{id}`

---

## 🧠 How It Works

1. Write code in editor  
2. Send request to backend  
3. Backend executes code  
4. Output returned instantly  
5. Save snippets if needed
